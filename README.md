# To-Do List Web App 

A simple and interactive To-Do List web application built to practice frontend development concepts.

## Overview
This project allows users to manage their daily tasks through a clean and simple interface.

## Features
- Add new tasks
- Edit tasks
- Delete tasks
- Mark tasks as completed
- Organize daily activities easily

## Technologies Used
- HTML
- CSS
- JavaScript
- ES6
